March 2024

This analysis was to explore the top industries and applicants in March 2024. The first graph is a visualization of applicants by school year, including advanced degrees and alumni. The second graph visualizes the total applicants by gender.

Code 
sy_longer <- march_school_year |> 
  pivot_longer(
    cols = alumni:sophomore,
    names_to = "school_year",
    values_to = "applicants",
    values_drop_na = TRUE
  )

sy_longer |> 
  mutate(school_year = school_year |> fct_infreq() |> fct_rev()) |> 
  ggplot(aes(school_year, fill = school_year)) +
  geom_bar(show.legend = FALSE)+
  scale_color_brewer(palette = "Set1")+
  labs(
    x = "",
    y = "",
    title = "Number of Applicants by School Year",
    subtitle = "March 2024"
  )
Code 
march_longer <- march_gender |> 
  pivot_longer(
    cols = female:male,
    names_to = "gender",
    values_to = "applicants",
    values_drop_na = TRUE
  )

march_longer |> 
  drop_na(job_type) |> 
  mutate(job_type = job_type |> fct_rev()) |> 
  ggplot(aes(job_type, fill = gender)) +
  geom_bar(position = "dodge")+
  scale_color_brewer(palette = "Set1")+
  labs(x = "",
       y = "",
       fill = "Gender",
       title = "Applicants by Job Type and Gender", 
       subtitle = "March 2024")

Applicants by Industry

This was completed to get a snapshot of engagement during the end of the fall semester.

Code 
applications <- my_postings 


applications |>
group_by(industry) |> 
  summarize(total = sum(applications_count)) |> 
  arrange(desc(total)) %>% 
  slice_head(n = 20) %>% 
  ungroup() %>% 
  mutate(industry = fct_reorder(industry, total)) %>% 
  ggplot(aes(industry, total, color = industry, size = total)) +
  geom_point() +
  coord_flip() +
  scale_y_continuous(labels = comma) +
  theme(legend.position = "none") +
  labs(x = "Industry",
       y = "Number of applicants",
       title = "Total Student Applicants by Industry",
       subtitle = "November 2024 to February 2025")

Employer Industries

This graph shows employer engagement by industry during the end of the fall semester. It reflects the number of job and internship postings by industry, versus student applications by industry as above.

Code 
top_industries <- my_postings

top_industries %>% 
 count(industry, sort = TRUE) %>% 
  slice_head(n = 20) |> 
  mutate(industry = fct_reorder(industry, n)) |> 
  ggplot(aes(industry, n, fill = industry))+
  geom_col()+
  coord_flip()+
  theme(legend.position = "none")+
  labs(title = "Top Employer Industries",
       subtitle = "November 2024 to February 2025", 
       x = "Industry",
       y = "Postings count")
